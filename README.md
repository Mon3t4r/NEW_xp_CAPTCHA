# xp_CAPTCHA(ddddocr白嫖版)


修改为使用ddddocr识别，对python版本没有原先那么多的要求了 目前尝试过3.7 3.8 3.9 

可事先使用ddddocr识别相关验证码，查看识别效果再选择是否使用该ocr


### 安装

环境需求：python3（ddddocr支持啥就能用啥。。） ###  ddddocr   #### jython（已包含在仓库中，不用网上到处找啦）

```
pip install ddddocr
```

python库只需要ddddocr这一个

### 安装

需要python3 小于3.7的版本

安装 `muggle_ocr` 模块（大概400M左右）
```
python3 -m pip install -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com muggle-ocr
```
